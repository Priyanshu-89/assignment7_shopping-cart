#  React Shopping Cart App

A simple and interactive **Shopping Cart** application built with **React.js**.  
It includes product listing, add-to-cart functionality, quantity control, toast notifications, and live total price updates.

---

##  Features

 Add and remove items from the cart    
 Display total price and number of items dynamically  
 Toast popups using **React Toastify**
